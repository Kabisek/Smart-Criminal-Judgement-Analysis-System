# Quick Start Guide

Get the JSON to CSV Converter running in 2 minutes!

## 1. Install Dependencies

```powershell
pip install -r requirements.txt
```

## 2. Run the Application

```powershell
python app.py
```

You should see:
```
 * Running on http://127.0.0.1:5000
```

## 3. Open in Browser

Open your web browser and go to:
```
http://localhost:5000
```

## First Conversion

1. Click **"Load Sample JSON"** on the right panel
2. Click **"Convert to CSV"** button
3. See your data converted and stored!

## Download Your Data

Use the download buttons in the "Data Management" section:
- 📄 **CSV** - For spreadsheet applications
- 🔗 **JSON** - For integration with other systems
- 📊 **Excel** - For advanced Excel features

## Key Features

✅ **Auto-formatting** - Click "Format JSON" to clean messy JSON
✅ **Instant preview** - See converted data immediately
✅ **Persistent storage** - All data saved automatically
✅ **Easy management** - View, delete, and download entries
✅ **Real-time stats** - Track your conversions

## Stop the Server

Press `Ctrl+C` in the terminal where the app is running.
